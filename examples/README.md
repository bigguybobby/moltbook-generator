# MoltBook Generator Examples

This directory contains example MoltBook listings generated by the tool.

## Examples

### CLI Tool
`example-cli-tool.yaml` - A Python CLI tool with automation features

### Library
`example-library.yaml` - A Rust async networking library

## Generate Your Own

Try generating listings for popular open-source projects:

```bash
# Python CLI
moltbook-gen https://github.com/astral-sh/ruff -o ruff-moltbook.yaml

# JavaScript Framework
moltbook-gen https://github.com/vercel/next.js -o nextjs-moltbook.yaml

# Rust Library
moltbook-gen https://github.com/tokio-rs/tokio -o tokio-moltbook.yaml

# Go CLI
moltbook-gen https://github.com/cli/cli -o gh-moltbook.yaml
```

## Tips

- Use `--preview` to review output before saving
- Add `--no-ai` for faster generation without AI descriptions
- Use `--verbose` to debug parsing issues
- Override auto-detection with `--tool-type <type>`
